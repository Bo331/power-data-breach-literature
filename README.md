## 🎯 本项目核心技术方案简介

为了让访问仓库的学者或评审专家快速获取干货，以下为您精炼总结本项目攻克的两大核心课题：

### 1. 电力专用的柔性自愈拦截 ── “温柔拦截（Gentle Containment）”
传统 IT 场景面临数据泄露时通常采用“直接断开网络”的粗暴阻断手段。但在电力网中，此举将导致设备心跳和 SCADA 调度命令丢失，极易引发大面积停电事故。
本项目提出并验证了**温柔拦截（柔性限速）策略**：
*   **限速机制**：通过在网关部署流量整形（Traffic Shaping），将异常主机的出站带宽限制在 **10 Kbps**；
*   **成效表现**：使 1GB 敏感配置文件（如变电站 SCD）外传耗时被强制拖长至 **9.9 天**，给应急响应争取了充裕时间；同时确保小于 1KB 的关键 SCADA 遥控命令与设备心跳实时流转，实现了**“阻断泄露，调度不断”**的电力安全微平衡。

### 2. 基于“网络-物理”共仿真的成效评估靶场
自动安全响应剧本（Playbooks）直接作用于电网一次侧及二次侧设备。为了在不干扰生产网的前提下实证其有效性，本项目汇编并论证了如下数字孪生验证技术：
*   **SG-ML 编译器**：基于 IEC 61850 标准配置文件（SCD/SCL），实现变电站数字孪生沙箱的秒级一键重构。
*   **PowerRange 测试床**：采用 Wattson 仿真网络通信，配合 pandapower 仿真物理电网，在网络拓扑中实现攻击重放（如 Industroyer 断路器操作），动态观察物理电网的潮流与电压波动，科学度量自动化响应（MTTD/MTTR）的物理保全平稳度。
